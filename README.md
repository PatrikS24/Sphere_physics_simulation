A program that simulates how spheres collide with each other and how their gravities effect each others paths through space.
Written in C++ using the Glew and Free Glut graphics api for OpenGL, Dear Imgui for the gui and cdelvs vector library is used for vectors (https://github.com/cdelv/Vector3D).
